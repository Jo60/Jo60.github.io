About Me

Introduction

I’m a freelance Front-End Web Developer looking to utilize my skills in a full-time position in the field of web development and design. I studied at George Brown College in 2017 where I learned Front-End web technologies such as HTML, CSS, JavaScript, jQuery, WordPress, React and Redux. I’m knowledgeable with SEO best practices and tools such as Google Analytics and AdSense. I’ve familiar with RESTful APIs to create web applications and hosting services like Netlify and Bluehost.

I’ve studied UX Design at Juno College in 2021. There I learned usability testing, in-context research and how to create Personas and Mental Models from that research. As well as becoming proficient with the prototyping tool Figma. Additionally, I learned to do Heuristic analysis and Accessibility Audits to create sites that are accessible and easy to use for the widest possible range of users. On every project I work on I’m always sure the aspects I’m in control of are compliant with the Web Content Accessibility Guidelines (WCAG) and the Accessibility for Ontarians with Disabilities Act (AODA).

I’ve gained experience not only by freelancing and with internships but through contract jobs as well. During these opportunities I’ve worked while adhering to industry standards and best practices. As a frontend web developer, I've built a website for a proposed university, creating the prototype using Bootstrap and custom CSS, since then I remade the site using React. Some of my freelance work also includes building a website for a Toronto Tax Firm from the ground up (check out the link below )) and improving the SEO and User experience of a Mississauga Construction Equipment Company’s WordPress site.

Photography is another skill I have. Took an introductory course in college and fell in love with it. I enjoy taking photos of landscapes and city structures like the ones you’ll see below and on my Instagram account. Professionally I've done Product photography for Instagram and Shopify accounts for small businesses.
